Real-Time Translation of Road Signs App

Cross-platform mobile application implemented using Python, React, Expo, and Flask API. Application detects, predicts, and translates road signs to aid drivers on the road by utilising YOLOv8 and a custom-built CNN model. The road sign description displayed to the user can be translated into other languages by using LibreTranslate API and the app can be run on iOS or Android devices.
